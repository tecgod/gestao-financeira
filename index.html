📖 Como configurar — Passo a passo

Crie uma planilha no Google Sheets e copie o ID
Abra sheets.google.com, crie uma nova planilha em branco e copie o ID dela da URL (guarde esse ID, ele será colado no campo "ID da Planilha" da tela de Configurações):
https://docs.google.com/spreadsheets/d/SEU_ID_AQUI/edit
Crie o Apps Script
Na planilha, clique em Extensões → Apps Script. Cole o código abaixo e salve:
/* Cole TODO este código no Apps Script */ const SENHA = "1234"; /* ← mude aqui */ const ABA_LANC = "Lancamentos"; const ABA_CAT = "Categorias"; function doGet(e){ return handle(e); } function doPost(e){ return handle(e); } function handle(e){ /* Lê payload base64 (GET) ou postData (POST) */ let body = {}; try { if(e.parameter && e.parameter.payload){ body = JSON.parse(decodeURIComponent(escape(Utilities.newBlob( Utilities.base64Decode(e.parameter.payload)).getDataAsString()))); else if(acao==='salvarVarios') res = salvarVarios(ss, body.itens); else if(acao==='excluirLancamento') res = excluirLancamento(ss, body.id); else if(acao==='editarLancamento') res = editarLancamento(ss, body.id, body.dados); } return ContentService .createTextOutput(JSON.stringify(res)) .setMimeType(ContentService.MimeType.JSON); } function garantirAbas(ss){ let al = ss.getSheetByName(ABA_LANC); if(!al){ al = ss.insertSheet(ABA_LANC); al.appendRow(['id','data','descricao','categoria','tipo','valor']); } let ac = ss.getSheetByName(ABA_CAT); if(!ac){ ac = ss.insertSheet(ABA_CAT); ac.appendRow(['Categoria']); ['Vendas','Serviços','Salário','Fornecedores', 'Aluguel','Contas','Marketing','Outros'] .forEach(c => ac.appendRow([c])); Publique como Web App
No Apps Script, clique em Implantar → Nova implantação.
Escolha tipo App da Web.
Em "Executar como" selecione Eu.
Em "Quem tem acesso" selecione Qualquer pessoa.
Clique em Implantar e autorize. Copie a URL gerada.
Cole a URL e o ID aqui e salve
Volte na tela de Configurações deste sistema, cole a URL do Web App e o ID da Planilha (do passo 1) nos campos acima e clique em Salvar configuração. Depois clique em Testar conexão para confirmar que está tudo certo.
Hospede no GitHub Pages (opcional)
Crie um repositório público no GitHub, faça upload deste INDEX.html e ative o GitHub Pages nas configurações do repositório (Settings → Pages → Branch main). Seu sistema ficará disponível em qualquer dispositivo com internet!
